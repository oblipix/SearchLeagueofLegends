# Como Clonar este Repositório

>
### Pré-requisitos:


> Ter o Git instalado no seu computador.
> Se ainda não tiver, baixe e instale o Git.
>
> Uma conta no GitHub (opcional, mas recomendado).
>
> **Fork do Repositório**
Um "fork" é uma cópia de um repositório que fica no seu perfil GitHub. Você faz um fork para ter uma versão própria do projeto na qual você pode trabalhar. Isso permite que você modifique e experimente o código sem afetar o repositório original.



**Vamos clonar este repositório passo a passo.**

_Obtenha a URL do repositório:_

`URL do repositório: https://github.com/oblipix/SearchLeagueofLegends.git`




navegue até a pasta onde deseja clonar.


**Clone o repositório:**

`git clone https://github.com/oblipix/SearchLeagueofLegends.git`


<sub> Agora você tem uma cópia local do repositório que pode ser modificada conforme necessário. </sub>
